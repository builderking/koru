# Support

Koru has no supported application release yet. The repository currently contains the product and engineering build plan.

When implementation begins:

- use GitHub Discussions for questions, ideas, compatibility reports, and help;
- use GitHub Issues for reproducible bugs and approved work;
- use private vulnerability reporting for security or privacy problems;
- do not include real prompts, clipboard contents, credentials, client information, or other private writing in any public report.

A versioned support and compatibility policy will be published before the first public beta.
