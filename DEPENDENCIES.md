# Dependencies and acknowledgments

Koru currently has no declared third-party runtime or development package dependency. Apple system frameworks are listed in `config/dependencies-allowlist.json`; they are platform APIs, not bundled third-party packages.

When dependencies are introduced, maintainers must commit the resolution, update the allowlist and this acknowledgment, review license/security/privacy/network/maintenance impact, and include the resolved manifest and CycloneDX SBOM in tagged releases. Apache-2.0 compatibility is not assumed merely because a package is open source.
